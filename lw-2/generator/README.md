Script, that creates ionic modules according to Merit structure.

Run it via nodejs, or call `lw-2/gm.sh`

Expample: 

`node run.js /some/awesome-stuff`
will create following structure:
```
src/
  app/
    some/
      awesome-stuff/    
        awesome-stuf.html
        awesome-stuff.module.ts
        awesome-stuff.scss
        awesome-stuff.ts 
```
