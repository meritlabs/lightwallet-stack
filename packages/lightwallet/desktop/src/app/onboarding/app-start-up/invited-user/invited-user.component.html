Creating your First Wallet

If you you have already been invited to the Merit world, then you can start using the Merit Web Wallet in a few simple clicks.

Simply click Create Wallet Below, and you will be able to enter an "Invite Code" and an "Alias."

The invite code should have been provided by the referring friend..

Your Alias is how you are known to other members of the Merit community, and is also your invite code.

*NOTE:* You will not be able to add or change your alias later; so it is recommended that you choose one now.
