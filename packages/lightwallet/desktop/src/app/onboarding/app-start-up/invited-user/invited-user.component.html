Create your first wallet

If you already have person who can invite you into the Merit World, you can start using your Merit Web Wallet in a couple of clicks.

Before you start, we recommend that you read this short tutorial.

After that you will press "Create Wallet" you will see screen with two fields: "Invite code or username" and "Your username (alias)"

* Be careful when you creating your Alias (user name) because it can be never changed.

Your alias is your invite code if you decide to invite somebody to Merit.
