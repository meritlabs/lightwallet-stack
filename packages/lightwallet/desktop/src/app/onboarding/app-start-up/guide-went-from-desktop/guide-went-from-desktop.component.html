Coming from a Desktop Wallet

If you have a Merit Desktop Wallet already, synchronizing it with or adding a separate Web Wallet is a great way to make your experience smoother.

There are 2 ways to get started with the Web Wallet.

Sync your desktop wallet to the web wallet.

To synchronize your web wallet with an existing wallet, you must first get your wallet's Pass Phrase.

Go to: Help > Debug Window > Console
Type: getmnemonic

Create a Standalone Web Wallet

To create a standalone wallet, you simply use your existing alias or address as your invite code. This method requires a spare invite that you must send to your new wallet in order to active it.

Ensure you have at least 1 available invite.
Use your own Alias as a referral.
