How to sync the Web and Desktop wallets

1. Download Desktop Wallet

2. Get your 12-word passphrase from the web wallet.

Click on your wallet and then click "backup" at the top of the sub-menu
Then click "Mnemonic phrase"

3. Run your "Desktop Wallet" client

Select the "Import Wallet" option
Enter your 12-word passphrase into the "import" text box

4. Wait until the wallet is done syncing the blockchain

5. Start Security Mining!
