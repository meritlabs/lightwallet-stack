How to sync Web Wallet between Desktop Wallet

1. Download Desktop Wallet

2. Go to the Web Wallet and export your wallet from the "Web Interface"

Choice needed wallet and select "backup" item
Then select "Mnemonic phrase"

3. Run your "Desktop Wallet" client

Select "Import wallet" item
Copy your "Mnemonic phrase" and paste into "Import box"

4. After successful wallet import, you should wait before your client will sync with block chain

5. After successful block chain sync you able to mine Merit

Go to application toolbar and select "Mining" item
Start mining
