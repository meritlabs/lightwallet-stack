If you will decide to create new wallet, you should do same actions (choose way to export your wallet and carefully save backup data).

For that, you should do several clicks:

  1. Select menu item Wallets (placed in the side bar)
  2. Choose wallet which you want to backup
  3. In the wallet details route select menu item Backup
  4. Choose the best way to you from available methods: QR Code, Mnemonic Phrase and a Backup file
